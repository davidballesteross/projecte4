# Producte 03: Identitat Corporativa, E-commerce i Sostenibilitat

## Descripció del producte
Aquest producte final reuneix les activitats creatives i estratègiques del Projecte 4 relacionades amb **UX/UI, disseny web, prototipat digital i sostenibilitat**.

Inclou el desenvolupament d’un **e-commerce complet a Figma**, la creació d’una **Landing Page comercial**, el prototipat del **procés de compra** i totes les reflexions i avaluacions de sostenibilitat del bloc final.

Aquest producte reflecteix competències transversals tant tècniques com creatives.
