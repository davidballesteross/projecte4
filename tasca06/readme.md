# Tasca 06: RDP (Remote Desktop Protocol)

## Descripció de la tasca
En aquesta tasca configurarem l’accés remot gràfic mitjançant **RDP**, tant a Windows com a Linux (Zorin OS).  
L’objectiu és crear una **guia de suport remot** que podrà utilitzar un futur becari d’EverPia per connectar-se ràpidament als equips dels clients.

La documentació ha de ser clara, visual i fàcil de seguir.

---

## Fases del projecte

### **Fase 1: Configuració RDP a Windows**
- Activar “Escriptori remot”.  
- Permetre connexions.  
- Validar l’accés des d’un altre equip.

### **Fase 2: Configuració RDP a Linux (Zorin OS)**
- Activar l’escriptori remot de GNOME.  
- Comprovar compatibilitat amb clients RDP.  
- Test de connexió.

### **Fase 3: Guia d’ús i resolució d’incidències**
- Com connectar-se pas a pas.  
- Errors típics i solucions.  
- Recomanacions de seguretat.

---

## Metodologia de treball
La tasca és individual.  
Cal fer una **PoC real** utilitzant màquines virtuals.  
La guia ha d’estar pensada per un client que necessita rapidesa i claredat.
